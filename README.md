# 🫧 BubblePop

Um experimento interativo de Front-end, com o objetivo de praticar conceitos de javascript e css criando uma experiência relaxante de estourar bolhas enquanto demonstra conceitos de manipulação do DOM e animações CSS.

## 🚀 Tecnologias
- **HTML5**: Estrutura semântica.
- **CSS3**: Animações com `@keyframes`, Flexbox e variáveis CSS.
- **JavaScript (Vanilla)**: Lógica de geração procedural de elementos e controle de eventos.

## 🧠 O que eu pratiquei neste projeto?
- **Manipulação Dinâmica do DOM**: Criação e remoção de elementos em tempo real para otimização de memória.
- **Física Simples com CSS**: Uso de funções matemáticas aleatórias para definir velocidade e tamanho das partículas (bolhas).
- **Gerenciamento de Estado**: Criação de um contador de pontuação que reage às interações do usuário.
